# **App Name**: CrossPay Dashboard

## Core Features:

- Dashboard Overview: Display overview cards for balances and quick actions on the dashboard.
- Wallet Management: Manage wallets with a list and a create wallet form. All info is client-side, stored in Redux.
- Currency Deposit Form: A form to simulate depositing funds.
- Currency Swap Form: A form to simulate currency swapping.
- Send Funds Form: A form to simulate sending funds to another wallet.
- Transaction History: Display a table of transaction history (sortable and paginated) using dummy data.
- Predictive Analytics Tool: Uses an AI model to predict future exchange rates. The tool will decide, using its internal reasoning, if those rates should be presented, and how.

## Style Guidelines:

- Primary color: Vivid blue (#468BFF) to represent trust and stability in financial transactions.
- Background color: Light blue-gray (#F0F4F8) for a clean, professional look.
- Accent color: Deep purple (rgb(34,26,54)) for interactive elements and call-to-action buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, neutral design.
- Code font: 'Source Code Pro' for displaying code snippets.
- Consistent use of financial and currency icons throughout the application to aid recognition.
- Responsive design utilizing TailwindCSS grid system for optimal viewing across desktop and mobile devices.
- Subtle animations and transitions to enhance user experience without being distracting.
